#### 1. Case 1: Sentiment Analysis Classification
Fokus pada sistem klasifikasi otomatis untuk mengecek dan melabeli kelas sentimen (Positif, Negatif, Netral).
* **Metode:** Text Preprocessing (Deep Cleaning), Feature Extraction (TF-IDF).
* **Model:** Logistic Regression, Support Vector Classifier (SVC), dan Random Forest.
* **Tujuan:** Membangun mesin yang mampu mengidentifikasi kelas sentimen pada data tweet baru secara otomatis dan akurat.

#### 2. Case 2: Clustering & Paslon Insight
Fokus pada analisis mendalam mengenai sebaran sentimen antar pasangan calon (Paslon) dan pengelompokan pola data.
* **Metode:** Clustering Analysis, Transformers-based analysis, dan Sentiment Interaction.
* **Tujuan:** Menemukan pola interaksi (seperti lonjakan 10rb+ tweet pada hari tertentu) untuk memahami respons publik terhadap masing-masing kandidat.
